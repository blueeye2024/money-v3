# 실매매_전략지침2.md

본 문서는 '안티그래비트(Antigravity)' 분석 시스템의 **SOXL 및 SOXS 전용** 실전 매매 전략과 운영 원칙을 정의합니다.

## 1. 전략 대상 및 데이터 지침
* **대상**: SOXL (Bull), SOXS (Bear) / **지표**: UPRO (Market Index)
* **주기**: 5분봉, 30분봉 (Extended Hours 포함)
* **데이터**: `market_candles` DB 증분 업데이트 및 KIS API 실시간 연동

## 2. 진입 전략 (Triple Filter)
다음 3가지 조건을 모두 충족 시 **100% 매수** 실행:
1. **30m Trend**: 30분봉 SMA 10 > SMA 30 (정배열)
2. **Day Power**: 현재가 >= 전일 정규장 종가 대비 +2%
3. **5m Timing**: 5분봉 SMA 10이 SMA 30을 골든크로스

## 3. 청산 및 리스크 관리 (Exit & Warning)
매수 후 다음 조건 발생 시 독립적으로 실행 :

### 3.1 수익 보존 및 절대 손절
* **Hard Stop**: 진입가 대비 **-8.5%** 도달 시 **100% 매도**
* **Trailing Stop**: 
    - 수익률 **+10%** 도달 시 활성화
    - 활성화 후 최고점 대비 **3% 하락** 시 **100% 매도**

### 3.2 단계별 경보 시스템
* **🟡 Yellow**: 5분봉 데드크로스 발생 시 → **보유량 30% 매도**
* **🟠 Orange**: 현재가 < 진입가 발생 시 → **보유량 30% 매도**
* **🔴 Red**: 30분봉 데드크로스 발생 시 → **잔량 100% 매도**

### 3.3 시장 지표(UPRO) 기반 청산
* **주의**: UPRO 30분봉 데드크로스 시 → **보유 주식 50% 매도**
* **위험**: UPRO 당일 고점 대비 -10% 하락 시 → **보유 주식 100% 매도**

## 4. 운영 및 UI 지침
* **SMS 제한**: 동일 종목 30분 내 중복 알림 금지 (DB 이력 관리)
* **시간 표기**: 🇺🇸 (NY) / 🇰🇷 (KR) 이중 표기 (`yyyy.MM.dd HH:mm`)
* **UI 상태**: 대기 시 Dark/Grey, 활성 시 Red(SOXL)/Blue(SOXS) 및 Pulse 효과
