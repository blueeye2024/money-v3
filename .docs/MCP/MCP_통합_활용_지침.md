# 🚀 MCP 서버 통합 활용 및 최적화 지침

## 1. 개요
이 문서는 Antigravity 시스템에 연동된 **18개의 MCP 서버**를 유기적으로 결합하여 최적의 성능을 발휘하기 위한 전략과 활용법을 기술합니다. 각 서버는 독립적으로 동작하지만, 함께 사용될 때 강력한 시너지를 발휘합니다.

## 2. 서버별 핵심 강점 및 활용 포인트

### 🎨 디자인 및 UI 개발 (Figma + Tailwind + Shadcn)
- **Figma**: 디자인 시안의 레이아웃, 색상 코드, 텍스트 스타일을 정확히 추출합니다.
- **Tailwind CSS**: Figma에서 추출한 값을 실제 CSS 유틸리티 클래스로 변환 및 추천합니다.
- **Shadcn/UI**: 변환된 스타일을 적용하여 즉시 사용 가능한 React 컴포넌트 코드를 생성합니다.
- **✨ 시너지 전략**: 
  > "피그마의 '로그인' 프레임을 분석해서(Figma), Tailwind 클래스로 변환하고(Tailwind), Shadcn의 Card와 Input 컴포넌트를 써서 구현해줘(Shadcn)"

### 💻 코드 및 인프라 관리 (GitHub + Docker + Shell)
- **GitHub**: 이슈 관리, 코드 리뷰, PR 생성을 담당합니다.
- **Docker**: 개발 환경 구축, 데이터베이스 컨테이너 관리, 배포 이미지 빌드를 수행합니다.
- **Shell**: 로컬 파일 조작 및 시스템 명령을 실행합니다.
- **✨ 시너지 전략**: 
  > "현재 변경 사항을 GitHub PR로 올리고(GitHub), 테스트용 DB 컨테이너를 Docker로 띄워서(Docker), 마이그레이션을 실행해줘(Shell)"

### 📊 데이터 분석 및 정보 관리 (Notion + Google Sheets + QuickChart)
- **Notion**: 프로젝트 문서화, 히스토리 관리, 지식 베이스를 구축합니다.
- **Google Sheets**: 대량의 데이터 처리, 재무 계산, 로그 저장을 담당합니다.
- **QuickChart**: 처리된 데이터를 시각적인 차트 이미지로 변환합니다.
- **✨ 시너지 전략**: 
  > "구글 시트의 매출 데이터를 읽어서(Sheets), 월별 추이 그래프를 그리고(QuickChart), 이를 노션 주간 보고서 페이지에 첨부해줘(Notion)"

### 🌐 정보 탐색 및 브라우저 제어 (Brave + Yahoo + Browser)
- **Brave Search**: 최신 기술 트렌드 및 해결책을 검색합니다.
- **Yahoo Finance**: 실시간 금융/주식 데이터를 조회합니다.
- **Browser/Puppeteer**: 웹 페이지 자동화, 스크린샷 캡처, 복잡한 UI 테스트를 수행합니다.
- **✨ 시너지 전략**: 
  > "Yahoo Finance에서 TSLA 주가를 조회하고(Yahoo), 관련 뉴스 기사를 검색해서(Brave), 해당 기사 페이지를 스크린샷 찍어서 저장해줘(Browser)"

### 🧠 지능형 사고 및 기억 (Sequential Thinking + Memory + Context7)
- **Sequential Thinking**: 복잡한 문제를 단계별로 분해하여 논리적 오류를 방지합니다.
- **Memory**: 중요한 사용자 선호도나 프로젝트 문맥을 장기 기억으로 저장합니다.
- **Context7**: 전체 소스코드의 문맥을 이해하고 최적의 코드를 제안합니다.
- **✨ 시너지 전략**: 
  > "이 기능 구현이 복잡한데 단계별로 생각해주고(Sequential), 과거에 내가 선호했던 방식(Memory)을 참고해서 전체 코드 맥락에 맞게(Context7) 수정해줘"

## 3. 최적 성능을 위한 팁

1.  **명확한 역할 부여**: 요청 시 "GitHub에서 이슈를 찾아서...", "Figma를 참고해서..." 처럼 사용할 도구를 명시하면 더 정확하게 작동합니다.
2.  **연쇄 작업 요청**: 한 번에 하나의 도구만 쓰는 것이 아니라, 위 시너지 전략처럼 여러 도구를 연결해서 요청하세요.
3.  **컨텍스트 유지**: Memory 기능을 활용하여 "이건 다음에 또 쓸 거니까 기억해둬"라고 말해두면 효율이 올라갑니다.

## 4. 원클릭 세팅 (One-Click Setup)

프로젝트 루트의 `setup_mcp.sh` 스크립트를 실행하여 이 모든 환경을 한 번에 재구성할 수 있습니다.

```bash
chmod +x setup_mcp.sh
./setup_mcp.sh
```
