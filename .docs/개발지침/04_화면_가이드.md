# 화면 표시 가이드

## 1. System Auto-Trading Log (Virtual)

### 개요
이 섹션은 시스템이 포착한 주요 매매 신호(Signal)를 기록하여 보여주는 리스트입니다.
사용자 요청에 따라 가상 매매 기록 대신 **실제 포착된 골든크로스/데드크로스 신호**를 우선적으로 표시합니다.

### 표시 데이터 소스
- **Signal History (30분/5분)**: `regimeDetails.soxl.cross_history`
- **표시 항목**:
  1. **Gold 30m**: 30분봉 기준 골든크로스 (강력 매수 신호)
  2. **Gold 5m**: 5분봉 기준 골든크로스 (단기 매수 신호)
  3. **Dead 5m**: 5분봉 기준 데드크로스 (매도/청산 신호)

### UI 구성
- **날짜/시간**: NY(뉴욕) 시간 기준 `MM/DD HH:mm` 포맷
- **가격**: 신호 발생 시점의 종가 (`$XX.XX`)
- **타입**: `⚡ SIGNAL` (골든크로스) / `💤 EXIT` (데드크로스)

## 2. 퍼포먼스 리포트 (System Performance)
- 상단 `System Auto-Trading Log` 헤더 아래에 위치.
- **표시 조건**: 실제 시스템에 의해 체결된 가상 매매(`trade_history`) 데이터가 존재할 때만 표시됩니다.
- 데이터가 없으면 자동으로 숨겨집니다.
