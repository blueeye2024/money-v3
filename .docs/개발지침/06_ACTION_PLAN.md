# 청안(Cheongan) Prime Guide : Action Plan

## 1. 현재 시스템 분석 (System Status V3.6)
**MASTER CONTROL TOWER** 완성을 통해 SOXL/SOXS에 대한 정교한 신호 포착 및 수동 제어 시스템이 갖춰졌습니다.

### ✅ 달성된 성과 (Achievements)
*   **Triple Filter System**: 30분봉 추세 + 박스권 돌파 + 5분봉 타이밍의 유기적 결합.
*   **V2 Hybrid Control**: 자동 신호 감지와 사용자 수동 개입(Signal Override)의 조화.
*   **Data Integrity**: 중복 기록 방지(30분), 실매매 기록(Real Trade) 분리 저장.
*   **UX/UI**: 직관적인 타워형 대시보드 및 SweetAlert2 기반의 깔끔한 상호작용.

### ⚠️ 발견된 개선 필요 사항 (Gap Analysis)
1.  **자금 관리(Money Management) 부재**:
    *   현재는 신호(Signal)만 줄 뿐, "얼마나(Qty)" 사야 하는지에 대한 가이드는 사용자 판단에 의존.
    *   *Risk Management* 모듈이 필요함 (예: 자산의 10% 진입, 손익비 계산).
2.  **백테스팅(Backtesting) 검증 부족**:
    *   현재 로직(Triple Filter)이 과거 데이터에서도 유효했는지에 대한 통계적 근거 부족.
3.  **알림 채널의 한계**:
    *   SMS(Twilio)는 비용이 발생하고 단방향임. Telegram/Slack 등 양방향 봇 연동 고려 필요.
4.  **확장성 제한**:
    *   현재 SOXL/SOXS에 하드코딩된 로직. 다른 종목(TSLA, TQQQ 등)으로 확장 시 코드 수정 필요.

## 2. 단기 액션 플랜 (Short-term: Ver 3.7 ~ 3.9)

### [P1] 포트폴리오 비중 제안 (Position Sizing)
*   **목표**: 신호 발생 시 "매수 추천 수량" 자동 계산.
*   **로직**: `(총 가용 예수금 * 설정 비중) / 현재가`.
*   **구현**: `User Config` 페이지에 '종목별 투자 비중' 설정 추가.

### [P2] 알림 시스템 고도화 (Advanced Alert)
*   **목표**: 텔레그램 봇(Telegram Bot) 연동.
*   **기능**:
    *   실시간 신호 수신.
    *   채팅으로 수동 신호 명령 전송 (예: `/buy SOXL`).
    *   현재 상태 조회 (예: `/status`).

### [P3] 모바일 전용 UI (Mobile First)
*   **목표**: 외부에서도 스마트폰으로 편하게 조작 가능하도록 CSS 최적화.
*   **내용**: 반응형 그리드, 터치 친화적 버튼 크기 조정.

### [P4] 피라미딩 전략 (Pyramiding)
*   **목표**: 추세 추종 시 "불타기" 가이드 제공.
*   **기능**: 수익 구간에서 추세 강화(Step 2->Step 3) 시 추가 진입 시점 및 수량 추천.
*   **Risk**: 평단가 관리 및 손익분기점(BEP) 자동 계산 시각화.

### [P5] 뇌동매매 방지 (Circuit Breaker)
*   **목표**: 감정적 매매 제어.
*   **기능**: 일일 손실 허용치(예: -5%) 초과 시 경고 또는 일정 시간 주문 잠금.

## 3. 중장기 로드맵 (Long-term: Ver 4.0 'Quantum')

### 🚀 AI 기반 시장 판단 (Market Sentiment Analysis)
*   LLM(Large Language Model) 연동.
*   뉴스 헤드라인, 공포-탐욕 지수(Fear & Greed)를 분석하여 Triple Filter의 민감도 조절.
*   예: "시장이 극도로 불안할 때는 Filter 3(30분봉) 확인 전까지 진입 금지".

### 📊 자동 백테스팅 엔진 (Auto-Backtester)
*   과거 1년치 데이터로 현재 전략 시뮬레이션.
*   승률(Win Rate)과 손익비(Profit Factor) 대시보드 제공.
